## ✨ Features

- **Glassmorphism UI:** A modern, sleek, and translucent design for a premium feel.
- **Fully Responsive:** Optimized for all devices, from large monitors to mobile phones.
- **Project Showcase:** Detailed cards for my featured projects with direct links to GitHub and Live demos.
- **Coding Stats:** Dynamic display of my LeetCode (1699 Max Rating) and CodeChef (2★) achievements.
- **Functional Contact Form:** Integrated with **Formspree** and **AJAX** for real-time email delivery without page reloads.
- **Clean Architecture:** Separated HTML, CSS, and JavaScript for better maintainability.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Flexbox & Grid), JavaScript (ES6+)
- **Backend Service:** Formspree API (for contact form)
- **Deployment:** Netlify / GitHub Pages
- **Icons & Fonts:** Font Awesome, Google Fonts (Plus Jakarta Sans)
